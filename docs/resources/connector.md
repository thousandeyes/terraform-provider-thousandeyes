# thousandeyes_connector (Resource)

Manages a ThousandEyes connector. A connector defines where webhook notifications are sent, including the target URL and authentication configuration.

## Example Usage

```terraform
resource "thousandeyes_connector" "example" {
  name   = "Example Connector"
  target = "https://webhook.site/your-id"

  headers {
    name  = "Content-Type"
    value = "application/json"
  }

  authentication {
    type     = "basic"
    username = "user"
    password = "pass"
  }
}
```
